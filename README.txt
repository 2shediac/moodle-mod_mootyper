This is the README file for the SiTyper project. SiTyper is a moodle extension, 
that simulates the 'typing instructor' programs. SiTyper is a free and open
source software. 

1. REQUIREMENTS

Plugin uses javascript which is not welcome in Moodle but it is a requirement for 
the use of the plugin. It creates 3 new tables in a moodle database and inserts a 
few sample typing exercises. This is all done automaticaly through the intstallation.

2. INSTALLATION

SiTyper is an activity module. You have to extract it to the mod directory. Than
go to Site Administration -> Notifications and click on the button.

3. USE

Using SiTyper plugin is very simple. An instance can be added as new activity in a
course like Lesson or Quiz. This video shows how to add exercise, create an instance
